# Stephcom Factory

**Stephcom Factory** est un atelier IA open source, local-first, conçu pour aider les non-experts à transformer leurs idées en projets numériques structurés, documentés, testables et vendables.

## Ligne directrice

- Open source et gratuit au maximum.
- Compatible Mac + Windows.
- Mistral réservé aux livrables à valeur commerciale.
- Les agents préparent, structurent et vérifient.
- L'humain garde le contrôle.
- Objectif court terme : vendre les premiers produits PDF premium.

## Projets prioritaires

1. `ASTRO_PREMIUM` — thème astral premium PDF personnalisé.
2. `NATURO_VITALITE` — bilan de vitalité personnalisé PDF prudent.

## Structure

- `app/` : future application locale.
- `core/` : agents, workflows, templates, prompts, qualité, providers IA.
- `modules/` : modules métier réutilisables.
- `workspace/` : projets, exports, archives.
- `data/` : base locale et réglages.
- `docs/` : documentation stratégique.
- `scripts/` : scripts Mac/Windows.

## Règle fondamentale

Aucun appel Mistral sans validation humaine et sans objectif commercial clair.
