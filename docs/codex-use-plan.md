# Plan d'utilisation Codex / OpenAI Credits

## Angle du dossier

Stephcom Factory est un framework open source local-first pour aider les non-experts, créateurs indépendants et petites équipes à transformer leurs idées en projets numériques structurés, documentés et vendables grâce à des agents IA.

## Utilisation prévue de Codex

- Génération de composants React.
- Backend Node/SQLite.
- Tests unitaires.
- Revue de pull requests.
- Correction de bugs.
- Documentation.
- Workflows agents.
- Sécurité basique.
- Refactoring.

## Pourquoi les crédits sont utiles

Ils accélèrent le développement open source du socle commun et des modules réutilisables : PDF premium, questionnaire, agents, templates projet.
