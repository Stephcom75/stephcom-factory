# Politique Tokens — Mistral

## Règle principale

Mistral ne sert pas à tester. Mistral sert à produire.

## Autorisé

- Rapport Astro final.
- Bilan Naturo final.
- Page de vente.
- Email client.
- Texte PDF premium.
- Audit qualité final.
- Correction critique d'un livrable commercial.

## Interdit

- Brainstorming.
- Tests aléatoires.
- Variantes sans objectif.
- Prompts flous.
- Discussions internes.
- Essais techniques inutiles.

## Workflow obligatoire

1. Préparation gratuite.
2. Prompt propre.
3. Validation humaine.
4. Génération Mistral.
5. Sauvegarde.
6. Contrôle qualité.
