# Workflow — De l'idée à la V1

1. Résumer l'idée.
2. Définir la cible.
3. Définir le problème résolu.
4. Définir le produit vendable.
5. Verrouiller la V1.
6. Lister ce qui est exclu.
7. Définir l'architecture.
8. Définir les données.
9. Définir le rendu premium.
10. Créer les prompts.
11. Tester.
12. Corriger.
13. Documenter.
14. Livrer.
